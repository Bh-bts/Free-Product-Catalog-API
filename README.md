# 📦 Product Catalog API

A simple and free FastAPI application to manage your product catalog!
You can add, update, delete, view, register, and login users.

🚀 Features

- Add new products
- Get all products
- Get single product
- Update product (Full PUT and Partial PATCH)
- Delete product
- User Registration
- User Login
- Validation and error handling

## ⚙️ Tech Stack

- FastAPI (Framework)
- SQLite (Database)
- SQLAlchemy (ORM)
- Pydantic (Data validation)

## 📥 Installation

Clone the repository

```commandline
git clone https://github.com/Bh-bts/Free-Product-Catalog-API.git
```

Install dependencies

```commandline
pip install -r /path/to/requirements.txt
```

Run the server

```
uvicorn main:app --reload
```

Open your browser and go to:

```commandline
http://127.0.0.1:8000/docs
```
